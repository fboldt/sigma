Ortomosaicos

O que é um ortomosaico?

Ortofotomosaicos são resultantes da montagem de imagens aerofotogramétricas ortorretificadas. São úteis para a identificação e mapeamento de feições geográficas e do uso do solo, entre outras informações que servem de base para orientar as ações institucionais e as políticas públicas na gestão ambiental.

Grupo SIGMA

O grupo SIGMA tem como objetivo o desenvolvimento de soluções para automatizar a geração de ortomosaicos, padronizando processos e reduzindo etapas manuais.

Image Stitching

O que é o Image Stitching?

Image stitching é a técnica de unir diversas imagens com áreas de sobreposição para formar uma única imagem contínua, desempenhando um papel fundamental na produção de ortofotomosaicos.

Fluxo de processamento

  1. Coleta de imagens
  2. Análise das imagens
  3. Ortoretificação
  4. Posicionamento das imagens
  5. Mosaico e fusão
